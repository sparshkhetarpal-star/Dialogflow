## 🌟 Key Features

* **NLU Powered:** Uses Natural Language Understanding to parse user financial data.
* **Loan Validation:** Automated logic to determine user eligibility.
* **Instant EMI:** Real-time calculation of monthly installments.
* **Guided UI:** Uses suggestion chips for a seamless user experience.

## 🛠️ Tech Stack

* **Platform:** Google Dialogflow (ES/CX)
* **Domain:** FinTech / Conversational AI
* **Concepts:** Intents, Entities, Contexts, and Fulfillment

---

## 🚀 Getting Started
1. Open the [Dialogflow Console](https://dialogflow.cloud.google.com/).
2. Create a new agent.
3. Import the agent ZIP file via **Settings > Export and Import**.
4. Test phrases like: *"What is my loan eligibility?"* or *"Calculate my monthly EMI"*.
